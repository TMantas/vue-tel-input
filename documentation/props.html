      # Props

      Property Type Default value Description
      allCountries Array All countries, see allCountries.js All countries passed to libphonenumber-js, can be overridden by this prop
      autoFormat Boolean true Auto update the input to the formatted phone number when it's valid
      customValidate Boolean, RegExp false Custom validation RegExp for input
      defaultCountry String '' Default country, will override the country fetched from IP address of user
      disabled Boolean false Disable vue-tel-input, including the input & flag dropdown
      autoDefaultCountry Boolean true To fetch default country based on IP address of user
      dropdownOptions Object Options for dropdown
      dropdownOptions.showDialCodeInList Boolean true Show dial code in the dropdown list
      dropdownOptions.showDialCodeInSelection Boolean false Show dial code in the dropdown selection
      dropdownOptions.showFlags Boolean true Show flags in the dropdown selection and list
      dropdownOptions.tabindex Number 0 Native dropdown tabindex attribute
      ignoredCountries Array [] List of countries will NOT be shown on the dropdown
      inputOptions Object Options for input
      inputOptions.autocomplete String 'on' Native input autocomplete attribute
      inputOptions.autofocus Boolean false Native input autofocus attribute
      inputOptions.id String '' Native input id attribute
      inputOptions.maxlength Number 25 Native input maxlength attribute
      inputOptions.name String 'telephone' Native input name attribute
      inputOptions.placeholder String 'Enter a phone number' Placeholder for the input
      inputOptions.readonly Boolean false Native input readonly attribute
      inputOptions.required Boolean false Native input required attribute
      inputOptions.tabindex Number 0 Native input tabindex attribute
      inputOptions.type String 'tel' Native input type attribute
      inputOptions.styleClasses String, Array, Object '' Custom classes for the input
      invalidMsg String ''
      mode String 'auto' Format number to 'international' (with + dial code) or 'national' (with 0...)
      onlyCountries Array [] List of countries will be shown on the dropdown
      preferredCountries Array [] Preferred countries list, will be on top of the dropdown
      styleClasses String, Array, Object '' Custom classes for the wrapper
      validCharactersOnly Boolean false Only allow valid characters in a phone number (will also verify in mounted, so phone number with invalid characters will be shown as an empty string)
